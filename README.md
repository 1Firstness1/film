# Список фильмов

React-приложение для управления персональной коллекцией фильмов с отслеживанием статусов просмотра.

## Краткое описание приложения

Веб-приложение "Список фильмов" позволяет пользователям эффективно управлять своей коллекцией фильмов. Вы можете добавлять фильмы, которые планируете посмотреть, отслеживать уже просмотренные и легко переключаться между этими статусами. Все данные сохраняются локально в вашем браузере.

## Основные возможности

- **Добавление фильмов** - внесение новых фильмов с названием и описанием
- **Управление статусами** - быстрое переключение между "Хочу посмотреть" и "Просмотрено"
- **Поиск по коллекции** - мгновенный поиск фильмов по названию и описанию
- **Визуальная статистика** - наглядное отображение количества фильмов в каждой категории
- **Локальное хранение** - все данные сохраняются в браузере и не теряются при перезагрузке
- **Адаптивный интерфейс** - удобное использование на компьютерах, планшетах и smartphones

## Особенности реализации

- **React 18** - современная версия React с использованием функциональных компонентов и хуков
- **Локальное состояние** - управление данными через useState и useEffect
- **Persistent storage** - автоматическое сохранение в localStorage браузера
- **Компонентная архитектура** - модульная структура с переиспользуемыми компонентами
- **Современный CSS** - Flexbox, Grid, градиенты, плавные анимации и переходы
- **Адаптивный дизайн** - корректное отображение на всех типах устройств
- **Доступность** - семантическая разметка и keyboard navigation

## Запуск приложения

### Предварительные требования
- Node.js (версия 14 или выше)
- npm или yarn

### Установка и запуск

1. Клонируйте репозиторий
2. Установите зависимости
3. Запустите приложение в development режиме
4. Откройте http://localhost:3000 для просмотра в браузере.

### Использование
**Добавление фильма:**
Введите название фильма в поле "Название фильма"
При желании добавьте описание
Нажмите "Добавить в список"
Изменение статуса:
Нажмите кнопку "Посмотрено" для фильмов в списке "Хочу посмотреть"
Нажмите "Не смотрел" для возврата фильма в список планируемых

**Поиск:**
Используйте поле поиска для быстрого нахождения фильмов

**Удаление:**
Нажмите кнопку "×" для удаления фильма из коллекции
